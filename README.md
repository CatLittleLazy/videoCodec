#### ODM 和 OEM

+ OBM：A设计，A生产，A品牌，A销售==工厂自己设计自产自销

+ ODM：B设计，B生产，A品牌，A销售==俗称“贴牌”，就是工厂的产品，别人的品牌

+ OEM：A设计，B生产，A品牌，A销售==代工，代生产，别人的技术和品牌，工厂只生产
